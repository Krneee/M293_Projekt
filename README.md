# M293_Projekt
Persönliche Portfolio-Website realisiert in HTML &amp; CSS mit responsivem Design, mehreren Unterseiten und Projektdetailseiten über mich selbst und meine zuletzt erarbeiteten Projekte

## Seiten
 
| Seite | Beschreibung |
|---|---|
| `index.html` | Startseite / Über mich |
| `faehigkeiten.html` | Technische Fähigkeiten & Kenntnisse |
| `projekte.html` | Übersicht aller Projekte |
| `kontakt.html` | Kontaktformular & Kontaktdaten |
| `projekt-reservetable.html` | Detailseite: Fullstack-Projekt ReserveTable |
| `projekt-yugioh.html` | Detailseite: React-Projekt YuGiOh Kartenbrowser |
| `projekt-schach.html` | Detailseite: Java-Projekt Schach |
 
## Technologien
 
- **HTML5**
- **CSS3** (CSS Variables, Flexbox, Grid)
- Responsive Design (Mobile-first)
- Google Fonts (Playfair Display, DM Sans)
