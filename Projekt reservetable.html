ReserveTable

Fullstack React Java SQL

Typ

Fullstack-Webapplikation

Jahr

2025

Frontend

React / HTML / CSS

Backend

Java (Spring Boot)

Beschreibung:

ReserveTable ist eine Fullstack-Webanwendung zur Tischreservierung in Restaurants. Nutzer können Restaurants durchsuchen, verfügbare Tische einsehen und Reservierungen direkt online vornehmen. Das System verwaltet Verfügbarkeiten in Echtzeit und sendet Bestätigungen an den Nutzer.

Meine Aufgaben:

Ich entwickelte das gesamte Frontend mit React sowie die REST-API im Backend mit Java. Das Datenbankschema wurde von mir in SQL entworfen und optimiert. Die Verbindung zwischen Frontend und Backend erfolgt über eine selbst entwickelte API.

Was ich gelernt habe:

Dieses Projekt hat mir geholfen, den gesamten Entwicklungsprozess einer Webapplikation zu verstehen, von der Datenbankmodellierung über die API-Entwicklung bis hin zur Benutzeroberfläche. Besonders wertvoll war die Erfahrung mit REST-APIs und der Kommunikation zwischen Front- und Backend.
