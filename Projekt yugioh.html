YuGiOh Kartenbrowser

React API JavaScript

Typ

React-Webapplikation

Jahr

2025

Framework

React

Daten

YGOPRODeck API

Beschreibung:

Der YuGiOh Kartenbrowser ist eine React-Webapplikation, die es Nutzern ermöglicht, die gesamte YuGiOh-Kartendatenbank zu durchsuchen und zu filtern. Karten können nach Name, Typ, Attribut und weiteren Kriterien gefiltert werden. Detailansichten zeigen alle relevanten Informationen zu jeder Karte.

Meine Aufgaben:

Ich integrierte die öffentliche YGOPRODeck-API und baute die gesamte Benutzeroberfläche in React auf. Dabei implementierte ich Suchfunktionen, Filterlogik und eine responsive Kartenanzeige mit Paginierung.

Was ich gelernt habe:

Dieses Projekt festigte meine Kenntnisse in React und im Umgang mit externen APIs. Ich lernte, Daten effizient zu laden, zu verarbeiten und benutzerfreundlich darzustellen. Besonders interessant war die Implementierung von Filterkomponenten und State-Management.
